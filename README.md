# Algorithm Design and Analysis (SC2001)

**NTU SC2001** - Lab Projects and Implementations

This repository contains the lab assignments and Jupyter Notebook implementations for the **SC2001 Algorithm Design and Analysis** course at Nanyang Technological University (NTU).

## 📚 Course Overview

The course covers fundamental concepts in algorithm design and analysis, including:
- Sorting algorithms and their analysis
- Graph algorithms (e.g., shortest paths)
- Dynamic Programming
- Divide and Conquer
- Greedy algorithms
- Complexity analysis (time & space)
